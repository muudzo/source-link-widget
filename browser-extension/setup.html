Welcome to Source Link Widget!

Follow these steps to get started

1 Start your Laravel server

Make sure your Laravel application is running:

php artisan serve

Your server should be running at http://localhost:8000

2 Run database migrations

Make sure your database is set up:

php artisan migrate
3 Create some categories (optional)

Visit http://localhost:8000/categories to create link categories.

4 Start saving links!

Click the extension icon in your browser toolbar to save the current page.
